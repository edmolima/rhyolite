# Packages & Templates

See the [Official Plugin Templates](./templates/plugins) for details on available templates.

## Official Packages

- **create-rhyolite**: The CLI tool for scaffolding projects
- **plugin**: Official plugin templates (React, Vue, Svelte, Solid, etc.)
- **themes**: Official theme templates

## Community Templates

Anyone can contribute templates! Community templates are found in:

- `packages/plugin/community/`
- `packages/themes/community/`

## Adding a Template

1. Fork the repo
2. Add your template JSON to the appropriate community folder
3. Submit a PR

See [Contributing](./contributing) for details.
