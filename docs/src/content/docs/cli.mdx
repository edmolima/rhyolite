# CLI Reference

See the [CLI Overview](./cli/index) for details on using the Rhyolite CLI.

- `--help`: Show help and usage

## Examples

```sh
# Create a plugin with React template
npm create rhyolite@latest my-plugin -- --plugin react

# Create a theme with Vue template
npm create rhyolite@latest my-theme -- --theme vue
```

See [Getting Started](./intro/getting-started) for more details.
