# Roadmap

Rhyolite is committed to delivering the best developer experience for Obsidian plugin and theme creators. Here’s what’s next:

## Near Term

- More official templates (Solid, Svelte, etc.)
- Improved docs and onboarding
- More automation in CI/CD

## Mid Term

- Plugin marketplace integration
- Advanced template customization
- Community template discovery

## Long Term

- Full-stack plugin support
- AI-powered code generation
- Deep Obsidian API integration

Have ideas? [Open an issue](https://github.com/edmolima/rhyolite/issues) or [start a discussion](https://github.com/edmolima/rhyolite/discussions)!
