# Getting Started

## Installation

```sh
# Scaffold a new plugin (React template)
npm create rhyolite@latest my-plugin -- --plugin react

# Scaffold a new theme (Vue template)
npm create rhyolite@latest my-theme -- --theme vue
```

## Requirements

- Node.js 18+
- pnpm (recommended)

## Local Development

```sh
git clone https://github.com/edmolima/rhyolite.git
cd rhyolite
pnpm install
pnpm build
```

For more, see the [Monorepo Guide](../monorepo.md).
