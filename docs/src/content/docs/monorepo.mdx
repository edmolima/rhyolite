# Monorepo Guide

See the [Monorepo Structure](./monorepo/index) for details on the workspace layout and rationale.

## Structure

- `packages/create-rhyolite`: CLI tool
- `packages/plugin`: Official plugin templates
- `packages/themes`: Official theme templates
- `packages/.../community`: Community templates

## Workspace Scripts

- `pnpm build`: Build all packages
- `pnpm test`: Run all tests
- `pnpm lint`: Lint all code

## Adding Packages

Add new packages or templates under `packages/` and update `pnpm-workspace.yaml`.

## Local Development

```sh
pnpm install
pnpm build
```

See [Contributing](./contributing) for more.
