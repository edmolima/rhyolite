## 2. Scaffold a Theme

```sh
pnpm create rhyolite@latest my-theme -- --theme minimal
```

## 3. Run & Test

Navigate to your new project and follow the README for next steps. Run tests and build with:

```sh
pnpm install
pnpm build
pnpm test
```
