# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing edmo.lima@example.com. We will respond as quickly as possible to address the issue.

Please do not disclose security issues publicly until they have been resolved.

## Supported Versions

| Version | Supported          |
| ------- | ----------------- |
| 1.x     | :white_check_mark:|

## Security Best Practices
- Keep your dependencies up to date
- Do not share sensitive information in public channels
- Use environment variables for secrets
